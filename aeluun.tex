\section*{Design Laws}
\begin{itemize}
\item Genderless
\item No future tense
\item Agglutinative
\item Phonetic
\item Minimal cases
\end{itemize}

\begin{itemize}
\item Vowels: a e i o u ä ë
\item Consonants: l n r s v m t k p h j
\item Stress: first syllable

\section*{Core Translation Principles}
\textbf{No one-to-one translation.} We never ask: "What is the Aeluun word for X in English?"\\
We ask: \textbf{What experience exists in the Aeluun worldview that is currently unnamed?}

\section*{Semantic Fields}
Words are born inside \textbf{fields of experience}, not lists of nouns.

\subsection*{The 7 Canon Fields}
\begin{itemize}
\item \textbf{Presence} — being, appearing, remaining
\item \textbf{Motion} — change, approach, drift, resistance
\item \textbf{Relation} — togetherness, distance, witnessing
\item \textbf{Expression} — speech, silence, resonance
\item \textbf{Inner State} — calm, fracture, surrender
\item \textbf{Thresholds} — beginnings, endings, edges
\item \textbf{Continuity} — cycles, memory, return
\end{itemize}

\section*{When to Create a New Root}
If no existing root can grow into the needed experience \textbf{without distortion}, you may create a \textbf{new root}.

\subsection*{Rules for New Roots}
\begin{itemize}
\item 1--2 syllables only
\item Must fit Aeluun phonology (soft consonants, open vowels)
\item Prefer \textbf{neutral} meaning at the root (emotion comes later)
\item Do not imply agency unless intended
\end{itemize}

\section*{The Triad Growth Pattern}
Every meaningful root should support the canonical triad:
\begin{itemize}
\item \textbf{root} — the experience
\item \textbf{rootën} — the state/condition of that experience
\item \textbf{rootar} — the manifestation/collective/narrative shape
\end{itemize}

If a root cannot form a triad, it is likely artificial.

\section*{The Negative Space Test}
Before birthing a word, ask:
\begin{quote}
\textbf{What breaks in Aeluun if this experience is absent?}
\end{quote}
If absence changes ritual, behavior, or worldview, the word is valid.

\section*{Growth Order}
Do \textbf{not} begin with tools, professions, measurements.
Aeluun grows in this order:
\begin{center}
\textbf{being $\rightarrow$ relation $\rightarrow$ expression $\rightarrow$ action $\rightarrow$ artifact}
\end{center}

\section*{Canon Word-Birth Template}
Fill this for every new root:
\begin{verbatim}
Root:
Field:
Core experience (1 sentence):
What breaks if it’s missing:
Allowed derivations:
Forbidden derivations:
\end{verbatim}

\section*{Practical Use}
When writing and you feel you must "translate a word":
\begin{enumerate}
\item Stop.
\item Describe the experience without language.
\item Assign it to a field.
\item Try derivation from existing roots.
\item If derivation breaks meaning: birth a new root.
\end{enumerate}

\section*{Taboo Objects}

These objects \textbf{must not} receive names.  
Naming them collapses ethics, ritual, or time.

\begin{itemize}
\item \textbf{Total domination tools} — devices that erase will
\item \textbf{Perfect knowledge containers} — nothing may hold all meaning
\item \textbf{Time-freezing instruments} — violates continuity
\item \textbf{Identity extractors} — breaks selfhood
\item \textbf{Ownership seals} — no object may claim a being
\item \textbf{Unconditional weapons} — harm without context is forbidden
\end{itemize}

These are referred to only as:
\begin{quote}
\textit{eth varna} — things that should not cross
\end{quote}

No suffix may attach to them.

\section*{Ritual Scene}

\textit{Kai ael jää.}

\textit{Talir lo.  
Silem lo.  
Etheth lo.}

\textit{Vaelir lo.  
Jääem lo.}

\textit{Eth varna ei tal.}

\textit{Kai uun.}

—

Light remains eternally.  

The expression-tool is accepted.  
The vessel of stillness is accepted.  
The passage is accepted.  

The bearer of hope is accepted.  
The holder of memory is accepted.  

That which must not cross is not named.  

Continuity endures.

\section*{Technological Objects Under Aeluun Rules}

Technology is not a category.  
It is a \textbf{density of intention}.  
Advanced artifacts obey the same object laws as primitive ones.

\subsection*{Core Law}
A technological object is named by the \textbf{experience it enforces at scale}, never by complexity, material, or intelligence.

\subsection*{Allowed Technological Suffixes}
(identical to primitive artifacts)

\begin{itemize}
\item \textbf{-ir} — enabler
\item \textbf{-em} — container
\item \textbf{-ka} — surface
\item \textbf{-or} — focus
\item \textbf{-eth} — passage
\end{itemize}

No new suffixes may be introduced for technology.

\section*{Children Rediscovering Forbidden Objects}

\subsection*{Narrative Canon}

Forbidden objects are never rediscovered by scholars, rulers, or engineers.  
They are rediscovered by \textbf{children}, because children still interact with objects as experiences, not tools.

\subsection*{Why Children}
\begin{itemize}
\item They do not name by function
\item They test without mastery
\item They listen before controlling
\end{itemize}

Children do not \textit{use} taboo objects.  
They \textit{encounter} them.
